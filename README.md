# 🔮 Glassmorphic Web Calculator

A sleek, beautiful, responsive web calculator designed with modern **Glassmorphism** UI principles. This project is built entirely using frontend technologies: HTML, CSS, and vanilla JavaScript. 

✨ **[Live Demo]**
    **(https://ganesh-nandam.github.io/my-calculator/)**
---

## 📸 Preview & UI Design

The calculator features a premium "frosted glass" interface, relying on transparency, background blur layers, and responsive interactive transitions.

*   **Frosted Glass Effect:** Powered by CSS `backdrop-filter: blur()`.
*   **Vibrant Backdrop:** A high-contrast linear gradient that beautifully showcases the glass transparency.
*   **Tactile Feedback:** Smooth hover and active click animations on every button.

---

## 🚀 Features

*   **Core Math Operations:** Addition, Subtraction, Multiplication, and Division.
*   **Instant Evaluation:** Real-time computation using optimized inline JavaScript execution.
*   **Smart Controls:** 
    *   `AC` for a complete screen clear.
    *   `DC` (Delete/Clear) to backspace a single digit.
*   **Fully Responsive:** Perfectly adapts to mobile, tablet, and desktop screens.

---

## 🛠️ Built With

*   **HTML5** - Structured semantics and button form layouts.
*   **CSS3** - Custom glassmorphic styling, flexbox layouts, and transition animations.
*   **JavaScript (ES6)** - Dynamic UI text updates and algebraic evaluations.
